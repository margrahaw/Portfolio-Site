Portfolio

Interactive Photo Gallery

A one page site built using HTML and CSS to create a searchable gallery of images. Using a jQuery plugin, lightbox.js was added to view large options of the images. In the image overlay, a short description of the image is provided based on image captions.

SVG Site Update

This project optimzied web performance by replacing images with SVGs. HTML and CSS were used to add inline SVGs and additional styles.

Interactive Video Player

This HTML5 video player has a custom controls bar built using JavaScript and the HTML5 Video Player. As the video plays, the transcript below the player is highlighted as the video progresses.

Web App Dashboard

An interactive dashboard for a web application that makes use of SVG graphics and JavaScript programing to create a multiple charts, graphs, and widgets.

Activity Guide For Utah National Parks

This website is a prototype for an informational site regarding National Parks in Utah. JavaScript was used to create a carousel and to access the Flickr API and create an image gallery of pictures of the various parks.

Spotify API Artist Search

A search app using Spotify's API. One can search for an artist and spotify will return a list of twelve album covers in a gallery. When a user clicks on an album cover, an image overlay with a larger image and ablum details will pop up..
